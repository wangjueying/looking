# 物品追踪器用户指南

这份文档面向 Android 用户，说明如何安装、配置和使用应用。

## 安装应用

1. 打开项目 Releases 页面：<https://github.com/wangjueying/looking/releases>
2. 下载最新的 APK 文件
3. 在手机上完成安装

如果系统阻止安装，请先在系统设置中允许安装未知来源应用。

## 首次配置

### 准备 API Key

应用使用 DashScope 进行识别，所以需要用户自行准备 API Key：

1. 访问 <https://dashscope.console.aliyun.com/>
2. 创建 API Key
3. 复制这个 Key

### 在应用内保存 API Key

1. 打开应用
2. 点击右上角设置按钮
3. 在 `API Key` 输入框中粘贴 Key
4. 返回首页开始使用

## 录入物品

首页支持三种录入方式：

- `拍照识别`
- `相册图片`
- `相册视频`

建议：

- 图片尽量清晰，避免主体被遮挡
- 视频尽量简短，镜头移动尽量平稳
- 对于多个物品的场景，尽量把关键物体完整拍进画面

## 搜索物品

1. 切换到 `搜索物品`
2. 输入关键词，例如物品名、描述词或位置词
3. 查看匹配结果
4. 点击结果进入详情页

详情页可查看：

- 原始图片或视频
- 识别出的名称、描述、位置
- 识别时间

## 管理物品清单

`物品清单` 页面显示当前保留在清单中的条目。

你可以：

- 浏览已保存的识别记录
- 点击进入详情页
- 将条目从清单中移出

需要注意：

- 从清单移出并不会删除底层识别记录
- 已移出条目仍然可以通过搜索找回
- 搜索结果中可以把条目恢复到清单

## 数据与隐私

- 识别时，图片或视频会发送给 DashScope API
- 识别完成后的结果默认保存在本地 SQLite 数据库
- 应用不会额外上传到项目自建服务器

## 常见问题

### 识别失败怎么办？

- 检查网络连接
- 检查 API Key 是否有效
- 尝试更清晰的图片或更短的视频

### 搜不到之前的物品怎么办？

- 尝试使用更短的关键词
- 确认该条目是否被移出清单
- 在搜索结果中查看是否带有 `已移出清单` 标记

### 清单里没有了，数据是不是丢了？

不是。当前设计是“移出清单但保留可搜索历史”，可在搜索页恢复。
